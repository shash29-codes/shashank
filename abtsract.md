# Abstract

In response to the growing concern over ATM thefts and fire-related incidents, an intelligent, real-time security system has been developed using embedded hardware and IoT technologies.  

The solution employs an **Arduino Uno microcontroller** to coordinate various components:
- **MEMS vibration sensor** for detecting physical tampering  
- **Infrared flame sensor** for identifying fire hazards  
- **GSM module** for sending instant SMS alerts to security personnel  
- **Buzzer** for audible alarms  
- **LCD screen** for real-time local display messages  

Unlike conventional ATM security systems that rely on surveillance footage and manual monitoring, this automated system provides immediate detection and response to potential threats.  

Key features:
- High sensitivity with low false alarm rates  
- Real-time notifications to reduce response time  
- Cost-effective and easily deployable, especially suitable for ATMs in remote or less-monitored areas
