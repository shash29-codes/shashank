# Design and Implementation of Theft Detection System at ATMs

This repository contains the **major project report** and **source code** for an IoT-based ATM theft detection system.  
Developed as part of the B.Tech program in Electronics and Communication Engineering at **Sree Chaitanya Institute of Technological Sciences (SCITS), Karimnagar**.

## 📖 Project Overview
- **Microcontroller:** Arduino UNO
- **Sensors:** MEMS vibration sensor, Flame sensor
- **Modules:** GSM for SMS alerts, LCD for local display
- **Features:** Real-time detection, low false alarms, cost-effective deployment

## 📂 Repository Contents
- `docs/` → Project report chapters in Markdown
- `src/` → Arduino source code
- `diagrams/` → Block diagrams, flowcharts, circuit diagrams
- `references/` → Citations and references

## 🚀 How to Run
1. Upload `src/main.ino` to Arduino UNO using Arduino IDE.
2. Connect GSM, MEMS, Flame sensor, and LCD as per circuit diagram.
3. Power the system and test vibration/fire detection.
